(Arquitecto Técnico_JMC) Porque no todo va a ser auditar modelos BIM e IFC. Este es un minijuego de supervivencia espacial desarrollado en un rato libre usando HTML5, Vanilla JS y TailwindCSS. Integra inteligencia artificial para darle un toque único a la narrativa de cada partida.

## 🚀 Características Principales

* **Mecánicas de Supervivencia:** Esquiva asteroides, naves piratas, cometas y singularidades que alteran la gravedad de tu nave.
* **Integración con IA (Gemini):** Los nombres de los sectores y el "Diario de Bitácora" final del capitán se generan dinámicamente en tiempo real usando la API de Gemini.
* **Power-Ups:** Recoge munición láser, escudos temporales, reductores de colisión y piezas para ensamblar un ataque EMP.
* **Físicas Nativas:** Atracción magnética, inercias, resoluciones de superposición y colisiones calculadas mediante código JavaScript puro, sin motores gráficos pesados de terceros.
* **Interfaz Futurista:** Diseño visual atractivo y minimalista apoyado en TailwindCSS, fuentes de Google Fonts (Orbitron) y animaciones CSS nativas.

## ⚙️ Instalación y Uso

Al ser un juego basado enteramente en el lado del cliente (Client-side), ponerlo a funcionar es inmediato.

1. Clona este repositorio:
   ```bash
   git clone [https://github.com/TU_USUARIO/jose-se-aburre-game.git](https://github.com/TU_USUARIO/jose-se-aburre-game.git)

2. Renombra el archivo JOSE SE ABURRE.txt a index.html.

3. Ábrelo directamente haciendo doble clic con cualquier navegador web moderno (Chrome, Firefox, Edge).

Nota: Para que el generador de reportes de IA funcione al final de la partida, es necesario insertar tu propia clave de la API de Gemini en la variable apiKey dentro del código.
