(Arquitecto Técnico_JMC) Porque no todo va a ser auditar modelos BIM e IFC. Este es un minijuego de supervivencia espacial desarrollado en un rato libre usando HTML5, Vanilla JS y TailwindCSS. Genera nombres de sectores e informes de misión de forma local, sin depender de ninguna API externa.

## 🚀 Características Principales

* **Mecánicas de Supervivencia:** Esquiva asteroides, naves piratas, cometas y singularidades que alteran la gravedad de tu nave.
* **Narrativa local:** Los nombres de los sectores y el diario de bitácora final se generan directamente en el propio navegador, sin necesidad de claves ni servicios externos.
* **Power-Ups:** Recoge munición láser, escudos temporales, reductores de colisión y piezas para ensamblar un ataque EMP.
* **Físicas Nativas:** Atracción magnética, inercias, resoluciones de superposición y colisiones calculadas mediante código JavaScript puro, sin motores gráficos pesados de terceros.
* **Interfaz Futurista:** Diseño visual atractivo y minimalista apoyado en TailwindCSS, fuentes de Google Fonts (Orbitron) y animaciones CSS nativas.

## ⚙️ Instalación y Uso

Al ser un juego basado enteramente en el lado del cliente (Client-side), ponerlo a funcionar es inmediato.

### Opción 1: abrir directamente en el navegador
1. Abre la carpeta [CODE](CODE).
2. Haz doble clic en [CODE/JOSE SE ABURRE.html](CODE/JOSE%20SE%20ABURRE.html).
3. Se abrirá en tu navegador web predeterminado.

### Opción 2: usar el lanzador de Python
1. Abre una terminal en la carpeta del proyecto.
2. Ejecuta:
   ```bash
   python CODE/JOSE\ SE\ ABURRE.py
   ```
3. El script abrirá automáticamente el archivo HTML del juego en tu navegador.

### Requisitos
- Un navegador moderno como Chrome, Edge o Firefox.
- Python 3 si prefieres usar el lanzador.

> Nota: El juego ya funciona sin ninguna API ni clave adicional. Todo el contenido narrativo se genera localmente.

### Controles
- Mueve el ratón o el dedo para dirigir la nave.
- Haz clic o toca para disparar.
- Recoge power-ups para mejorar tu supervivencia.
